The list above has class="demo" and the links in it were manipulated using the jquery-showLinkLocation plugin like this:

        // execute on page load
        $(function() {
                // for each anchor within some tag with class="demo"
                $('.demo a').showLinkLocation();
        });

As you see, the manipulated links have the URL appended.
